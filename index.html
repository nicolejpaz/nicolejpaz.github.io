Cofounder/Deployed

Fremote: Watch videos simultaneously!
Fremote is a web app to remote-control YouTube & Vimeo videos. A user can send a link to a friend(or friends) they are chatting with and they can watch the video/movie at the same time. Whoever has control of the "remote" can start/pause/fast-forward/rewind the video and it will control the video for every viewer.

Technologies used: Rails, JavaScript, MongoDB

Actually Has Code on Github

Diary Tracker: Have you ever had a medical issue where your doctor had you keep a diary? If so, do you remember bringing it back to the doctor and having them take their time to skim its entire contents? Well, this intends to solve that issue — at least in regard to sleep diaries — by having the user fill out a form once a day. It only checks for sleep length and quality, but I intend to include a textbox for the user to describe their sleep patterns. These are then put on a summary page, which the user can then show to their doctor. The textbox will take some more time because it'll be a word cloud on the summary page, and will eventually allow the user to sort through the cloud (i.e., only show words that appeared when the quality of sleep was ranked 'Zombie').
