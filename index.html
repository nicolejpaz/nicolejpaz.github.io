Actually Has Code on Github

Weather Alert: A weather alert app that sends a text to a user if there are any warnings issued by the National Weather service. They can choose on the website whether they want to get all of them, or only the ones labelled severe or extreme (yes, the NWS does have an extreme category, because weather can be really neat).

CardIt: An app that has a set of flashcards for the user to learn more about a subject. It will allow them to create their own flashcards. Am making it because my new car allows me to use the NATO phonetic alphabet to enter in an address in the navigation, and I'm too lazy to make flash cards (you know, because pens are just so heavy).

Currently Planning

Look Up Genetics: Ever since 23andme was forced to take away some of their functionality, there is a gaping hole that needs to be filled. This will allow doctors to easily look up any medical problems that may arise based off of a 23andme report. Reason for this: doctors have a hard time finding this information themselves, and it's important for a patient to know. My cousin is helping me find more information on the subject, and I'm currently scouring assorted online information and libraries in order to learn more about genetic testing.
